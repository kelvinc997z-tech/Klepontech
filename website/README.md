# 0x100x Clone – Crypto Research Landing Page

Website yang menyerupai gaya dan struktur **0x100x.com** – platform penelitian kripto untuk investor serius.

---

## 🚀 **Deploy Sederhana**

### **Lokal (Preview)**
1. Pastikan file berada dalam struktur:
   ```
   website/
   ├── index.html
   ├── style.css
   ├── script.js
   └── assets/ (opsional untuk gambar)
   ```
2. Buka `index.html` langsung di browser.

### **Netlify / Vercel ( Gratis )**
- Drag & drop folder `website/` ke Netlify
- Atau push ke GitHub + import di Vercel

### **Custom Domain**
Setelah deploy, setting CNAME ke domain Anda.

---

## 📝 **Customisasi**

- **Edit teks** di `index.html` (section Framework, Research, dll).
- **Ganti warna** di `style.css` (variabel CSS di bagian top).
- **Update link** ke `/the-record` dan `https://club.0x100x.com/join` sesuai kebutuhan.
